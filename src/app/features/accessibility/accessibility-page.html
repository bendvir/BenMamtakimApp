המחויבות שלנו

ממתקי התקווה שואפת להנגיש את האתר לכלל הלקוחות, לרבות אנשים עם מוגבלויות. אנו פועלים לעמוד ברמת הנגישות AA של תקן WCAG 2.1 ובדרישות תקן ישראלי IS 5568 לפי תקנות שוויון זכויות לאנשים עם מוגבלות.

פעולות נגישות שבוצעו באתר

  • ✓ תפריט נגישות — כפתור קבוע המאפשר שינוי גודל טקסט, ניגודיות גבוהה, גווני אפור, הדגשת קישורים, עצירת אנימציות וגופן קריא
  • ✓ שפת הדף — הוגדרה לעברית לתמיכה מלאה בקוראי מסך
  • ✓ ניווט מקלדת — כל פעולות האתר נגישות דרך מקלדת בלבד
  • ✓ תוויות ARIA — כל הכפתורים, הטפסים והאלמנטים האינטראקטיביים כוללים תיאורים לקוראי מסך
  • ✓ טקסט חלופי לתמונות — כל תמונות המוצרים כוללות תיאור טקסטואלי
  • ✓ סמנטיקת HTML5 — שימוש בתגיות header, main, nav, footer לניווט קל לקוראי מסך
  • ✓ קישור "דלג לתוכן" — מאפשר קפיצה ישירה לתוכן הראשי
  • ✓ ניגודיות צבעים — יחסי ניגוד העומדים בדרישות רמת AA
  • ✓ תמיכה ב-RTL — הדף מוגדר כולל לכיוון ימין-לשמאל (עברית)
  • ✓ שמירת העדפות — הגדרות הנגישות נשמרות בין ביקורים

מגבלות ידועות

אנו ממשיכים לשפר ולבדוק את הנגישות. מגבלות ידועות נכון לתאריך ההצהרה:

  • ! מסמכים PDF (אם קיימים) ייתכן שאינם נגישים במלואם
